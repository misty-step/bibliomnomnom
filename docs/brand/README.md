# Bibliomnomnom Brand Guidelines

This directory contains brand documentation for Bibliomnomnom.

## Contents

| File | Description |
|------|-------------|
| [colors.md](./colors.md) | Color palette, hex codes, usage rules |
| [typography.md](./typography.md) | Fonts, type scale, hierarchy |
| [voice.md](./voice.md) | Brand voice, tone, messaging guidelines |

## Quick Reference

### Colors
- **Background:** Bone `#FAF8F5`
- **Text:** Ink `#1C1917`
- **Accent:** Gold `#C5A059`

### Fonts
- **Headings:** Playfair Display (serif)
- **Body:** Geist (sans-serif)
- **Code:** JetBrains Mono

### Voice
- Warm, intellectual, privacy-focused
- Talk to "readers" not "users"
- Quietly confident, never shouty

## Usage

These guidelines ensure consistency across:
- Landing page and marketing site
- In-app UI copy
- Ad creatives (Google, Meta)
- Social media presence
- Email campaigns

## Design Tokens

The colors and typography defined here are implemented as design tokens in `/lib/design/tokens.generated.ts`. Always use the token system rather than hardcoding values.

## Questions?

For brand-related questions, check with the team or refer to the full guidelines in each file.
